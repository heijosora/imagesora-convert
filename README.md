# Sigma-Sora Convert

Web-based image processing and conversion tool built with vanilla JavaScript and Canvas API. Transform your images with advanced filters, resize them precisely, and convert between multiple formats - all in your browser, no uploads to servers required.

## Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS
- **Icons**: Heroicons
- **Image Processing**: Canvas API
- **Build Tool**: None (Vanilla JavaScript)
