# Sigma-Sora Convert

A modern, web-based image processing and conversion tool built with vanilla JavaScript and Canvas API. Transform your images with advanced filters, resize them precisely, and convert between multiple formats - all in your browser, no uploads to servers required.

2. Enable GitHub Pages in your repository settings

### Option 2: Local Development

   ```bash

   cd sigma-sora-convert
   ```

2. Open `index.html` in your web browser

### Option 3: Local Server
For the best experience with file uploads, serve the files using a local web server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Then open http://localhost:8000 in your browser
```

## üìñ Usage Guide

### Single Image Conversion
1. Navigate to the **Core Converter** page
2. Upload an image by dragging & dropping or clicking "Browse Files"
3. Adjust filters and size settings as needed
4. Select output format and quality
5. Click "Convert Image" and then "Download"

### Batch Processing
1. Go to the **Batch Processor** page
2. Select multiple images
3. Configure your settings
4. Process all images at once
5. Download as a ZIP file
